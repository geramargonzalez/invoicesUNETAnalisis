Machine Learning · Análisis Predictivo

Análisis Predictivo de Ingresos — Upwork

Modelo de regresión para estimar la tarifa horaria de freelancers en Upwork. Análisis de ~60 000 registros con ingeniería de características sobre skills, nivel de experiencia y localización de clientes. Proyecto del MIT Summer Program – Uruguay 2026.

EDA · Análisis de Datos Abiertos

Siniestros de Tránsito — Montevideo 2022

Análisis exploratorio completo sobre 7 802 siniestros de tránsito registrados en Montevideo durante 2022. Preprocesamiento robusto, corrección de calidad de datos y 13 visualizaciones que revelan patrones temporales, por tipo de vehículo, sexo y resultado del accidente.

7 802 registros 13 visualizaciones Datos abiertos IMM
Python Pandas Matplotlib Seaborn Plotly

Deep Learning · Segmentación Semántica

Segmentación de Mascotas con U-Net

Implementación de U-Net desde cero para segmentar píxeles del cuerpo de mascotas (primer plano / fondo / borde) sobre el Oxford-IIIT Pet Dataset. Alcanza 89.64 % Pixel Accuracy y Dice 0.831 con solo ~7.8 M parámetros entrenados en menos de 3 000 imágenes.

Accuracy 89.64 % Dice 0.831 Mean IoU 0.729
TensorFlow Keras U-Net Oxford-IIIT Pet Segmentación

Deep Learning · OCR · Document AI

Detección de Campos en Facturas — U-Net

Pipeline end-to-end para extraer información estructurada (empresa, fecha, dirección, total) de imágenes de facturas usando U-Net con backbone MobileNetV2 + EasyOCR. Demo interactiva en Gradio. Entrenado sobre el dataset ICDAR-2019-SROIE (626 recibos).

Pixel Acc. 92 % Mean IoU 0.41 Demo Gradio
TensorFlow U-Net MobileNetV2 EasyOCR Gradio
